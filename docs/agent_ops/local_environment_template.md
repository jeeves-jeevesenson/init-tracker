# Local Environment Template

This template shows what local-only environment details should be recorded in `docs/local/production_environment.md`.

## Production Server Details

- **LAN IP**: <e.g., 192.168.1.x>
- **Hostname**: <e.g., dnd.example.com>
- **SSH User**: <e.g., admin>

## Application Paths

- **App Path**: <e.g., /home/user/app>
- **Venv Path**: <e.g., /home/user/venv>
- **Logs Path**: <e.g., /home/user/logs>
- **Releases/Backups Path**: <e.g., /home/user/releases>

## Runtime Configuration

- **Port**: <e.g., 8787>
- **Runtime Command**: <e.g., /path/to/venv/bin/python3 serve_headless.py --host 0.0.0.0 --port 8787>
